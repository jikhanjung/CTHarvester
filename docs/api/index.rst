Core Modules
------------

The core modules provide the fundamental functionality for thumbnail generation,
progress tracking, and image processing.

UI Modules
----------

The UI modules contain all user interface components including dialogs, widgets,
and the main window.

Utility Modules
---------------

Utility modules provide helper functions and classes for common operations like
settings management, file validation, and resource management.

Configuration
-------------

Configuration modules define application constants, settings schemas, and
default configurations.

Security
--------

Security modules handle file validation, path sanitization, and secure file
operations to prevent directory traversal and other security vulnerabilities.
